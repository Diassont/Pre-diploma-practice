# DIS STORE — WordPress Theme

Власна тема WordPress для інтернет-магазину комп'ютерної техніки **DIS STORE**.  
Розроблена в межах переддипломної практики зі спеціальності «Інженерія програмного забезпечення».

---

## 🛍 Про проєкт

**DIS STORE** — повноцінний інтернет-магазин комп'ютерної техніки, побудований на платформі **WordPress + WooCommerce** з власною темою оформлення.  
Проєкт розроблявся локально у середовищі **LocalWP** і включає всі необхідні сторінки, шаблони та функціональність для реального інтернет-магазину.

---

## ⚙️ Функціональність

- Каталог товарів з категоріями, фільтрацією за ціною та текстовим пошуком
- Сортування товарів (за ціною, назвою, датою додавання)
- Картки товарів із підтримкою знижок і кастомних лейблів (ТОП, Хіт, Новинка тощо)
- Сторінка товару з галереєю зображень та вкладками: Опис, Характеристики, Доставка, Гарантія
- Додавання в кошик через AJAX без перезавантаження сторінки
- Інтеграція з **YITH Wishlist** (обране) та **YITH Compare** (порівняння товарів)
- Сторінки кошика та оформлення замовлення
- Особистий кабінет покупця
- Адаптивна верстка для мобільних пристроїв і планшетів

---

## 📦 Реліз (повна копія сайту)

У розділі **Releases** містяться файли, створені плагіном **Duplicator**:

- `installer.php`
- `archive.zip`

> ⚠️ Повна копія сайту (база даних + файли) запакована окремо та додана у розділ **Releases**.

---

## 🚀 Розгортання

1. Завантажити `installer.php` та `archive.zip` з розділу **Releases**
2. Розмістити обидва файли у кореневій папці сайту
3. Відкрити в браузері `http://ваш-сайт/installer.php`
4. Пройти покроковий майстер встановлення Duplicator
5. Сайт повністю готовий до роботи
